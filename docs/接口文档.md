# 极趣云平台 API 文档

## 概述

本文档描述极趣云平台（Zectrix）开放API的使用方法。

### 认证方式

所有API请求需要在Header中携带API Key：

```bash
-H "X-API-Key: zt_your_key_here"
```

### 基础URL

```
https://cloud.zectrix.com/open/v1
```

---

## 设备管理

### 获取设备列表

获取用户绑定的所有设备列表。

**请求**

```
GET /devices
```

**完整URL**

```
https://cloud.zectrix.com/open/v1/devices
```

**curl 示例**

```bash
curl \
  "https://cloud.zectrix.com/open/v1/devices" \
  -H "X-API-Key: zt_your_key_here"
```

**返回示例**

```json
{
  "code": 0,
  "data": [
    {
      "deviceId": "AA:BB:CC:DD:EE:FF",
      "alias": "我的设备",
      "board": "bread-compact-wifi"
    }
  ]
}
```

**返回字段说明**

| 字段 | 说明 |
|------|------|
| deviceId | 设备ID（MAC地址） |
| alias | 设备别名 |
| board | 设备型号/板型 |

## 待办事项

### 获取待办列表

获取用户的待办事项列表。

**请求**

```
GET /todos
```

**完整URL**

```
https://cloud.zectrix.com/open/v1/todos
```

**查询参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| status | integer | 否 | 过滤状态：0=待完成, 1=已完成 |
| deviceId | string | 否 | 设备ID(MAC地址)，过滤指定设备的待办 |

**返回示例**

```json
{
  "code": 0,
  "data": [
    {
      "id": 1,
      "title": "买牛奶",
      "description": "",
      "dueDate": "2026-03-20",
      "dueTime": "09:00",
      "repeatType": "none",
      "repeatWeekday": null,
      "repeatMonth": null,
      "repeatDay": null,
      "status": 0,
      "priority": 1,
      "completed": false,
      "deviceId": "AA:BB:CC:DD:EE:FF",
      "deviceName": "我的设备",
      "createDate": "2026-03-18 10:00:00",
      "updateDate": 1742284800
    }
  ]
}
```

**返回字段说明**

| 字段 | 说明 |
|------|------|
| id | 待办ID |
| title | 待办标题 |
| description | 待办描述 |
| dueDate | 截止日期 |
| dueTime | 截止时间 |
| repeatType | 重复类型：none=不重复 |
| repeatWeekday | 重复星期（如每周一重复） |
| repeatMonth | 重复月份 |
| repeatDay | 重复日期 |
| status | 状态：0=待完成, 1=已完成 |
| priority | 优先级 |
| completed | 是否已完成 |
| deviceId | 关联设备ID |
| deviceName | 关联设备名称 |
| createDate | 创建时间 |
| updateDate | 更新时间戳 |

### 创建待办

创建一个新的待办事项。

**请求**

```
POST /todos
```

**完整URL**

```
https://cloud.zectrix.com/open/v1/todos
```

**请求体**

JSON格式，包含待办信息。

**返回示例**

```json
{
  "code": 0,
  "data": {
    "id": 1,
    "title": "买牛奶",
    "status": 0,
    "priority": 1,
    "deviceId": "AA:BB:CC:DD:EE:FF",
    "createDate": "2026-03-18 10:00:00"
  }
}
```

---

### 更新待办

更新指定待办事项的信息。

**请求**

```
PUT /todos/{id}
```

**完整URL**

```
https://cloud.zectrix.com/open/v1/todos/{id}
```

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | integer | 是 | 待办ID |

**请求体**

JSON格式，包含需要更新的待办字段。

**返回示例**

```json
{
  "code": 0,
  "data": {
    "id": 1,
    "title": "买牛奶和面包",
    "status": 0,
    "priority": 1
  }
}
```

## 显示推送

### 推送图片到设备

将图片推送到设备的墨水屏显示。

**请求**

```
POST /devices/{deviceId}/display/image
```

**完整URL**

```
https://cloud.zectrix.com/open/v1/devices/{deviceId}/display/image
```

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| deviceId | string | 是 | 设备ID(MAC地址) |

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| images | file[] | 是 | 图片文件，支持多张(最多5张)，单张不超过2MB |
| dither | boolean | 否 | 是否使用抖动算法(默认true)，关闭则使用硬阈值二值化 |
| pageId | integer | 否 | 页面编号(1-5)，指定后会持久化存储 |

**返回示例**

```json
{
  "code": 0,
  "data": {
    "totalPages": 1,
    "pushedPages": 1,
    "pageId": "1"
  }
}
```

---

### 推送文本到设备

将文本内容推送到设备的墨水屏显示。

**请求**

```
POST /devices/{deviceId}/display/text
```

**完整URL**

```
https://cloud.zectrix.com/open/v1/devices/{deviceId}/display/text
```

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| deviceId | string | 是 | 设备ID(MAC地址) |

**请求体**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
text  string 是  文本内容(最多5000字)，支持换行
fontSize  integer 否  字体大小(12-48，默认20)
pageId  string  否  页面编号(1-5)，指定后会持久化存储


**返回示例**

```json
{
  "code": 0,
  "data": {
    "totalPages": 1,
    "pushedPages": 1,
    "pageId": "1"
  }
}
```

---

### 推送标题+正文到设备

将结构化文本（标题+正文）推送到设备的墨水屏显示。

**请求**

```
POST /devices/{deviceId}/display/structured-text
```

**完整URL**

```
https://cloud.zectrix.com/open/v1/devices/{deviceId}/display/structured-text
```

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| deviceId | string | 是 | 设备ID(MAC地址) |

**请求体**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
title string  否  标题文本(最多200字)，与body至少填一项
body  string  否  正文内容(最多5000字)，支持换行
pageId  string  否  页面编号(1-5)，指定后会持久化存储





**返回示例**

```json
{
  "code": 0,
  "data": {
    "totalPages": 1,
    "pushedPages": 1,
    "pageId": "1"
  }
}
```

---

### 删除页面

删除设备显示的指定页面，或清空全部页面。

**请求**

```
DELETE /devices/{deviceId}/display/pages/{pageId}
```

**完整URL**

```
https://cloud.zectrix.com/open/v1/devices/{deviceId}/display/pages/{pageId}
```

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| deviceId | string | 是 | 设备ID(MAC地址) |
| pageId | integer | 否 | 页面编号，不传则删除全部页面 |

**返回示例**

```json
{ "code": 0, "msg": "success" }
```
